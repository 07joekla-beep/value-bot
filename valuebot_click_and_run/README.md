# ValueBot v1 – “klicka och kör” (lokalt)

Du kan köra en lokal webbsida (Streamlit) genom att klicka på en fil.

## Windows
1. Packa upp zip.
2. Dubbelklicka **run_windows.bat**
3. Webben öppnas på `http://localhost:8501`

## Mac / Linux
1. Packa upp zip.
2. Kör i Terminal:
   ```bash
   ./run_mac_linux.sh
   ```
3. Webben öppnas på `http://localhost:8501`

## Riktiga odds
- Byt namn `config.example.json` → `config.json`
- Fyll i API-nyckel
- Välj **oddsapi** i sidomenyn.
